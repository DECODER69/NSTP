Steps to be followed when using this system.


Migrations
1. Create database with the name of  "nstpsystem"
2. python manage.py makemigrations nstpApp
3. py manage.py migrate


Create Superuser ( just type the provided code on ide terminal and follow the directions for creating superuser)
superuser is the only one who can access admin side of the program

4. python manage.py createsuperuser



Execution of the program(just type the provided code on ide terminal)
5. py manage.py runserver


Creating student account

6.Just open the regioster form on landing page of the program and create account, the login usong the provided ID Number and your password,.
